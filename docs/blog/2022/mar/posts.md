Это програма выстрели себе в ногу или умри

Сегодня мы разберем способ сломать себе линукс #8:
устанавливаем rofi
пишем скрипт в test.sh:
```bash
#!/usr/bin/env bash
if [ $@ ]; then
xdg-open $@
else
echo 1
fi
```

запускаем
```bash
rofi -show abobus -modi abobus:./test.sh
```

нажимаем Enter и готово! Вы великолепны!
